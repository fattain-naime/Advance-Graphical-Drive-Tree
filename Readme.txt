Advance Graphical Every Drive Tree provide you to tree you system every drive at one. you don't need edit or add new drive in code.

The program will automatically detected you system drive, HDD drive, SSD drive, external drive, CD drive (if a cd in there) also you don't need to install any software.

How to run
To run Advance Graphical Every Drive Tree just dabble click on AGT.cmd. To stop tree press Ctrl+c and type y and hit enter.
